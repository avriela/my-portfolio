О проекте
Бот предоставляет пользователям доступ к платному сервису через Telegram. После оплаты пользователь получает персональную ссылку для активации в клиентском приложении. Поддерживается несколько тарифных планов, пробный период, управление устройствами и реферальная программа. Администратор управляет всем через отдельную веб-панель.
Часть 1 — Telegram-бот
Запуск и главное меню
Команда /start открывает главное меню с кнопками: Купить подписку, Личный кабинет, Получить бесплатно, Реферальная программа, Инструкция по подключению, Поддержка, Наш канал.
Пробный период
Выдаётся один раз на аккаунт. При первом нажатии пользователь получает активационную ссылку и информацию об условиях пробного доступа. При повторном входе в раздел отображается реферальная программа и личная реферальная ссылка.
Тарифы и оплата
Доступно два тарифа с разными лимитами одновременных подключений. Каждый тариф имеет несколько периодов: от недели до года. После успешной оплаты бот автоматически выдаёт персональную ссылку для вставки в клиентское приложение.
Личный кабинет
Отображает Telegram ID, имя пользователя, текущий баланс, активный тариф и срок действия, а также персональную ссылку. Внутри доступны функции управления устройствами и перевыпуск ключа доступа.
Управление устройствами
Пользователь может просматривать список подключённых устройств, добавлять новые и удалять существующие в рамках лимита тарифа. Перевыпуск ключа обновляет ссылку — старая становится недействительной.
Реферальная программа
Пользователь получает личную реферальную ссылку. За каждого приглашённого начисляется бонус на баланс. Доступны: настройка реквизитов для вывода, отправка заявки на вывод и история всех предыдущих выводов.
Инструкции и поддержка
Раздел инструкций содержит кнопки по каждой поддерживаемой платформе. При нажатии открывается пошаговое руководство и ссылка на скачивание клиентского приложения. Кнопка поддержки ведёт в Telegram-канал.
Часть 2 — Веб-панель администратора
Пользователи
Общий список с Telegram ID, username, балансом, датой регистрации и статусом. Карточка пользователя показывает время последнего визита, все подписки, реферальную статистику, историю платежей. Доступно ручное пополнение баланса.
Подписки
Список всех подписок с ID, тарифом, количеством устройств, датами создания и истечения, статусом. В карточке подписки доступно продление на указанное количество дней и деактивация подписки.
Платежи
Список всех транзакций: ID платежа, имя пользователя, сумма, способ оплаты, дата и статус. Фильтрация и поиск по параметрам.
Рассылки
Создание рассылки с изображением и текстом, HTML-форматирование, предпросмотр перед отправкой. Фильтр аудитории: все пользователи, с активной подпиской, на пробном периоде, с истекшей подпиской. После отправки — статистика доставки.
Управление ботом
Разделы: тексты бота, кнопки меню, настройки, цены тарифов, инструкции по платформам. Все тексты и цены редактируются без участия разработчика.
